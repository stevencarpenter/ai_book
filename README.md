# The Welch Labs Illustrated Guide to AI

Supporting code for the [Welch Labs Illustrated Guide to AI](https://www.welchlabs.com/ai-book).

## Quick Start

### Prerequisites

- Python 3.10+
- (Recommended) An NVIDIA GPU with CUDA, or Apple Silicon Mac (MPS). CPU works for some chapters but is much slower.

### Setup

```bash
git clone https://github.com/stephencwelch/ai_book.git
cd ai_book
python -m venv .venv
source .venv/bin/activate   # On Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

Some notebooks (Chapters 2, 3, 8) use Meta's Llama 3.2 model, which requires a Hugging Face account with access granted:

1. Create an account at [huggingface.co](https://huggingface.co)
2. Request access to [meta-llama/Llama-3.2-1B](https://huggingface.co/meta-llama/Llama-3.2-1B)
3. Log in locally: `huggingface-cli login`

## External Data

### AFHQ Dataset (Chapter 1)

Required for `1_perceptron_cats_vs_dogs.ipynb`:

1. Download from [Kaggle: Animal Faces](https://www.kaggle.com/datasets/andrewmvd/animal-faces)
2. Extract into `data/afhq/` so the structure is:
   ```
   data/afhq/train/cat/
   data/afhq/train/dog/
   ```

### Pre-computed Loss Landscapes (Chapter 2)

Some visualization cells in `2_gradient_descent.ipynb` load pre-computed `.npy` files generated by running `2_wormhole.ipynb`. If you haven't run the wormhole notebook (it requires significant GPU time), you can skip those cells or place the pre-computed data in `data/wormhole_merged/` and `data/apr_29_2/`.
